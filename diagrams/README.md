
# Diagramas

Este diretório será utilizado para armazenar diagramas da arquitetura do laboratório, incluindo:

* Topologia do ambiente
* Fluxo Suricata → Wazuh
* Arquitetura SIEM/IDS
* Diagramas de comunicação entre componentes

Os diagramas servirão como apoio à documentação técnica e à compreensão da infraestrutura implementada.
